# GoF Design Patterns (TypeScript)

## Реалізовані патерни

### Creational
- Factory Method
- Builder

### Structural
- Adapter
- Decorator

### Behavioral
- Strategy

## Запуск
npx ts-node examples/run-all.ts

## Висновки
Патерни GoF покращують структуру коду, зменшують звʼязність і полегшують розширення проєкту.
